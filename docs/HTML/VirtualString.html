FixedSizeString is a fixed-size and null-terminated string. Since the size is a template parameter, each different size is a different object type. But since FixedSizeString is derived from VirtualString (which is not a templated object), different size instances can be interchanged through references to VirtualString. VirtualString implements almost all of std::string functionalities.
FixedSizeString does not do any dynamic memory allocation/deallocation.
FixedSizeString "Convenient typedefs" are among the types that DataFrame library can read/write from/to files and serialization
