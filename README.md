# DataFrame
This is a C++ statistical library to provide an interface similar to Pandas package in Python<BR>
For documentation, please see the **DataFrameDoc** file


# Installing using CMake
```
mkdir build
cd build
cmake ..
make install
```

# Uninstalling

```
cd build
make uninstall
```
