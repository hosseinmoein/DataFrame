template<typename T>
std::vector<typename T> &
create_column(const char *name);
It creates an empty column named “name”
Returns a reference to the vector for that column. If a column with the same name and type already exists, it retunrs a reference to it. 		T: Type of the column 
    MyDataFrame         df;
    std::vector<int>    &col0 =
        df.create_column<int>(static_cast<const char *>("col_name"));
